# Prompt Templates

Use these English templates as starting points. Replace bracketed fields with the user's details and keep any constraints the user provided.

## Poster

```text
A 16:9 cinematic poster for [subject/product/event], featuring [main visual], [style or genre], premium lighting, strong focal point, clean composition, space for title text, high-resolution editorial design.
```

## Cover

```text
A polished 16:9 cover image for [topic/product/channel], showing [main subject], [mood], crisp details, balanced negative space, modern professional layout, visually striking but not cluttered.
```

## Social Media Image

```text
A high-impact social media image for [platform/use case], featuring [subject], [brand mood/colors if any], clean background, strong visual hierarchy, sharp details, optimized for attention and readability.
```

## Product Image

```text
A premium product hero image of [product], [angle/view], realistic studio lighting, clean background, accurate shape and materials, refined reflections, commercial photography style, 16:9 composition.
```

## Image Editing

```text
Edit the input image to [desired change]. Preserve [subjects or details to keep unchanged]. Improve [lighting/background/composition] while keeping a natural, high-quality result.
```

## Text to Video

```text
A [duration]-second video of [subject/action], [camera movement], [scene/mood], [lighting], [style], smooth motion, coherent details, 16:9 aspect ratio.
```

## Image to Video

```text
Animate the input image into a [duration]-second video with [camera movement], subtle parallax, natural motion, consistent subject identity, no distortion of the main subject, 16:9 aspect ratio.
```

## Negative Constraints

When useful, append concise constraints:

```text
Avoid distorted text, warped faces, extra limbs, broken geometry, flickering, noisy artifacts, excessive blur, and changes to the main product shape.
```
